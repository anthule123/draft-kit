# Gõ kí hiệu toán học ở Latex và typst 

Latex
```
1. belongs to: \in 
2. (does not belong to): \notin 
3. union : \cup 
4. intersection: \cap 
5. proper subset: subset
6. subset: \subseteq
7. set difference: setminus 
8. emptyset: \emptyset 
9. Natural numbers: \mathbb{N} 
10. integers : \mathbb{Z }
```

Typst: 
```
1. belongs to: $in$
2. (does not belong to): $nin$ 
3. union : $union$
4. intersection: $sect$ 
5. proper subset: $subset.strict$
6. subset: $subset$
7. set difference: $setminus$ 
8. emptyset: $empty$ 
9. Natural numbers: $NN$ 
10. integers :      $ZZ$
```
